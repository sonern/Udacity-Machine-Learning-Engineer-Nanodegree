Libraries: This project uses python 3 Some key python libraries used are:

    ipython, jupyter: for development within notebooks
    numpy, scipy: as common and core scientific libraries
    pandas: for working with data frames & series
    sklearn: for machine learning functionality
    matplotlib, seaborn: for data visualisations

- Workbook can be run one buy one. 
- In order to calculate different kernels,svm kernel should be changed manually.
- For PCA section n_componenets should be set manually.
